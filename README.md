# Анализ данных сотрудников и заработных плат

## Описание проекта
Этот проект представляет собой полный цикл работы с HR-данными, включающий очистку, подготовку, анализ и визуализацию данных в Excel. Основная цель — продемонстрировать навыки аналитики данных, работу с Power Query, построение дашборда и формирование выводов на основе статистических показателей.


## Состав датасета
После подготовки данные включают следующие переменные:
- **ID сотрудника**
- **Возраст**
- **Возрастные категории** (<25, 25–35, 35–45, >45)
- **Страна**
- **Отдел**
- **Должность**
- **Зарплата**
- **Категории зарплат** (Low, Medium, High по перцентилям)
- **Дата трудоустройства**
- **Стаж в месяцах**
- **Стаж в годах**
- **Категории стажа** (<1 года, 1–5, 5–10, >10 лет)


## Используемые инструменты
- **Microsoft Excel**
  - Power Query для очистки данных
  - Сводные таблицы
  - Формулы и перцентильные расчёты
  - Построение графиков и дашборда
  - **BoxPlot** диаграммы (аналитика распределений)


## Структура проекта
Проект состоит из нескольких рабочих листов Excel:

### 1. **Dashboard**

### 2. **Анализ сотрудников**
- Распределение сотрудников:
  - по странам
  - по отделам
  - по должностям
  - по возрастным категориям
  - по стажу и категориям стажа
  - по зарплатным категориям
- Графики:
  - столбчатые
  - круговые
  - линейчатые

### 3. **Анализ заработных плат**
- Средние, медианные, минимальные и максимальные зарплаты:
  - по странам
  - по отделам
  - по должностям
  - по возрастным категориям
- Графики столбчатые

### 4. **BoxPlot**
- Общее распределение зарплат
- BoxPlot зарплат по категориям стажа

### 5. **Salary vs Experience**
- Создание финального графика зависимости зарплаты от стажа:
  - Средняя
  - Медианная
  - Минимальная
  - Максимальная

### 6. **employee_records**
- Импорт и предварительная очистка данных
- Удаление ошибок
- Добавление вычисляемых колонок
- Данные синтетические и часть значений была дополнительно модифицирована для демонстрации аналитических приемов
